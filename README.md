# Stefon Simmons Resume



## Install

Install and read my resume via npm.
#### in your command line (cl)
```bash
$ mkdir stefons-resume
$ cd stefons-resume
$ npm install stefonsimmons-resume
$ touch index.js
```

#### inside index.js:
```js
const resume = require("stefonsimmons-resume")
console.log(resume)
```
#### back in your cl
```bash
$ node index.js
```
